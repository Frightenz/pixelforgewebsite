# PixelForge Website

Static GitHub Pages site for PixelForge.

## Pages

- `index.html` - product landing page
- `privacy.html` - App Store privacy policy URL
- `terms.html` - terms of use
- `support.html` - App Store support URL

## Development

Open `index.html` directly in a browser or serve this folder with any static server.

Run checks:

```bash
npm test
```
